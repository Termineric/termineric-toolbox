### ðŸ”¹ Begrippenlijst

- **Variabele:** Opslagplaats voor data in je programma
- **Functie (Function):** Een stuk code dat iets doet en iets teruggeeft

- **NOOB:** Iemand die net begint met programmeren (dat is prima!)
